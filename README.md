# Меню дизайн-портфолио 🎨

Это простая консольная программа на Python, которая имитирует дизайн-портфолио.

## Возможности
- Навигация по меню
- Раздел с постерами
- Раздел с кино-постерами
- Раздел брендинга
- Счётчик просмотров
- Простая анимация загрузки

## Как запустить

1. Установите Python
2. Запустите файл:

python main.py

## Пример

=== DESIGN MENU ===
1 - Poster Design
2 - Movie Posters
3 - Branding
0 - Exit
